# wegame

wegame 是一个使用 Flutter 开发的应用程序。

A new Flutter project.

## 功能

- **文本转语音:**  用户可以输入文本，应用会将其转换为语音。
- **音频存储:** 生成的音频文件会存储在 PocketBase 中。
